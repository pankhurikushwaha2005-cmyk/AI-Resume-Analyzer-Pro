# 📄 AI Resume Analyzer Pro

## 🚀 Overview
AI-powered Resume Analyzer that helps job seekers improve their resume by analyzing ATS score, extracting skills, and matching with job descriptions using AI.

---

## ✨ Features
- 📊 ATS Score Calculator
- 📄 Resume Parsing (PDF/DOCX)
- 🛠 Skill Extraction
- 📋 Job Description Matching
- 🤖 AI Resume Suggestions

---

## 🛠 Tech Stack
- Python
- Streamlit
- NLP
- Git & GitHub

---

## 📌 Project Highlights
- Built an interactive web app using Streamlit  
- Extracts and analyzes resume content automatically  
- Matches resume with job descriptions  
- Provides improvement suggestions using AI logic
